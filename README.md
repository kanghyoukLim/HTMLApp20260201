# 로또 번호 추천 사이트

## 개요

1부터 45까지의 숫자 중 6개의 고유한 로또 번호를 추천하는 간단한 웹 애플리케이션입니다.

## 특징

*   **번호 생성:** 6개의 고유한 랜덤 번호를 생성합니다.
*   **표시:** 생성된 번호를 사용자에게 보여줍니다.
*   **사용자 인터페이스:** 간단하고 직관적인 인터페이스.

## 개발 계획

*   **HTML:** 제목, 버튼, 결과 영역으로 기본 구조를 만듭니다.
*   **CSS:** 더 나은 사용자 경험을 위해 페이지 스타일을 지정합니다.
*   **JavaScript:** 로또 번호 생성 및 표시 로직을 구현합니다.

# Lotto Number Recommendation Site

## Overview

This is a simple web application that recommends 6 unique lottery numbers from 1 to 45.

## Features

*   **Number Generation:** Generates 6 unique random numbers.
*   **Display:** Shows the generated numbers to the user.
*   **User Interface:** A simple and intuitive interface.

## Current Plan

*   **HTML:** Create the basic structure with a title, a button, and a result area.
*   **CSS:** Style the page for a better user experience.
*   **JavaScript:** Implement the logic for generating and displaying the lottery numbers.
